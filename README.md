📌 Project Title

Movie Ticket Booking System using Java

📖 Project Overview

The Movie Ticket Booking System is a Java-based desktop application developed to automate the process of booking movie tickets and managing theatre seat reservations.
This project provides users with a simple and user-friendly platform to view available movies, select seats, book tickets, and generate booking details digitally.
The system reduces manual work involved in traditional ticket booking methods and helps in managing movie schedules, seat availability, and customer bookings efficiently.
The project is developed using Java, Swing/AWT for GUI, and MySQL database for storing booking information.

🎯 Objectives of the Project

To automate movie ticket booking operations.
To reduce manual errors in seat allocation.
To provide fast and secure ticket booking.
To manage movie details and customer information efficiently.
To demonstrate Object-Oriented Programming concepts using Java.

✨ Features

🎥 View available movies
🪑 Seat selection and reservation
